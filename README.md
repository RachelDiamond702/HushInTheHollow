# Hush In The Hollow
Hush in the Hollow is an interactive board game set in a magical forest overtaken by a goblin town. Move carefully to avoid alerting the goblin army as you journey through the ever-changing forest. Collect treasures, defeat enemies, and emerge as the victor of the forest.

# How Gameplay Works
The board integrates Hall effect sensors and NeoPixel LEDs to create a dynamic, hidden-map system. Player pieces contain magnets that activate sensors embedded beneath each tile, triggering visual feedback when a tile is explored.

This prevents players from seeing the full map in advance and enables randomized board states for each playthrough. As a result, gameplay is driven by exploration, uncertainty, and adaptive decision-making.

# How It Was Made
* 3D Printing — Fabricated player standee bases and card slider mechanisms
* Laser Cutting — Constructed the board enclosure, player pieces, and paper environment elements (trees, houses)
* 3D Modeling — Designed enclosure structure, standee bases, sliders, and environmental components for fabrication
* Adobe Illustrator — Created game cards and formatted the ruleset
* Procreate — Illustrated card artwork and visual assets
* Electronics Integration — Implemented NeoPixel LEDs, Hall effect sensors, and servo motors to create a responsive, interactive board
