Hush in the Hollow

By: Rachel Diamond

About the Game

Hush in the Hollow is an interactive board game set in a magical forest overtaken by a goblin town. Move carefully to avoid alerting the goblin army as you journey through the ever-changing forest. Collect treasures, defeat enemies, and emerge as the victor of the forest.

How to Play

Objective

Be the first player to reach the maximum XP...

Setup

  • Each player chooses a Character Class Card.
  • All players begin at the outer edge of the board.
  • Shuffle all decks.

Turn Structure

  1. Roll the dice.
  2. Move.
  3. Resolve tile.

Tile Encounters

Enemy Tiles

  • Draw an Enemy Card and roll.
  • Success: Gain XP.
  • Failure: Weakened next turn.
  • Damage: Roll again.

Treasure Tiles

  • Draw a Treasure Card.
  • Success: Gain treasure.
  • Failure: Goes to bottom of deck.
  • Destroyed: Lost.

Resource Tiles

  • Draw a Resource Card:
    • Gold Coins
    • Potions
    • Armor / Weapons
    • Sellable Resources

Player Combat

  • Same tile = combat.
  • Set wager.
  • Roll dice.
  • Highest wins.

Goblin Town Event

Triggers global battle players can join or decline.

  • If you join:
    • Escaped
    • Injured
    • Captured
  • If you decline:
    • No effect

Winning the Game

First to max XP wins.
